# GeoASK
Project Description

GeoASK is an Android mobile application designed to support geoparticipatory surveys. The application allows users to explore a selected area on a map, select specific locations, and complete surveys evaluating those spaces. User responses are stored in a Firebase Realtime Database.

The application was developed using Kotlin with the Android SDK and Google Maps API.

Features: 
-application splash screen 
-home screen with instructions 
-list of available surveys 
-description page of the surveyed area 
-interactive map with polygon-based spatial areas 
-survey dialog triggered after clicking a map area 
-saving survey responses to Firebase database

Running the Project

Clone the repository:
git clone https://github.com/emiliaszulc/GeoASK

1. Open the project in Android Studio.
2. Add your Google Maps API key to AndroidManifest.xml.
3. Configure Firebase and add the google-services.json file.
4. Build and run the application on an emulator or Android device.
   
Opis projektu

GeoASK to aplikacja mobilna na system Android umożliwiająca prowadzenie ankiet geopartycypacyjnych. Użytkownik może przeglądać mapę wybranego obszaru, zaznaczać konkretne przestrzenie oraz wypełniać ankiety dotyczące jakości i funkcjonalności tych miejsc. Odpowiedzi użytkowników zapisywane są w bazie danych Firebase.

Aplikacja została napisana w języku Kotlin z wykorzystaniem Android SDK oraz Google Maps API.

Funkcjonalności: 
-ekran startowy z instrukcją korzystania z aplikacji 
-lista dostępnych ankiet -opis wybranego obszaru badawczego 
-interaktywna mapa z zaznaczonymi poligonami reprezentującymi analizowane przestrzenie 
-ankieta wyświetlana po kliknięciu obszaru na mapie 
-zapis odpowiedzi użytkownika w bazie danych Firebase.
